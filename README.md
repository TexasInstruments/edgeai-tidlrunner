# edgeai-tidlrunner

Easy to use commandline tool for "Bring Your Own Edge AI Models" (BYOM) - to work with TI Deep Learning (TIDL) for TI MPU Processors for Edge AI. 

There are two packages in this repository with different purposes.
1. [tidlrunner](tidlrunner): Model Compilation, Inference, Analysis or Benchmark. 
2. [tidloptimizer](tidloptimizer): Quantization Aware Distillation to make the models more friendly for TIDL. Sparsity/Prunning (In future). 

Please go through the documentation of [tidlrunner](tidlrunner) to get started with this repository. [tidloptimizer](tidloptimizer) is a work in beginning stages and documentation is yet to be added. 

Notice
* More details about TIDL and other tools are explained in [Edge AI developer landing page](https://github.com/TexasInstruments/edgeai) and [edgeai-tidl-tools](https://github.com/TexasInstruments/edgeai-tidl-tools) - recommend to read these.
